<h1 style="text-align: center">call center 后台管理系统</h1>

#### 项目简介

一个基于 Spring Boot 3 、 Spring Boot Jpa、的前后端分离的外呼call center管理系统

**重要：项目目前处于开发阶段，bug比较多**

**体验地址：**  暂无

#### 功能列表

- [x] 租户权限管理
- [x] 外呼拨打
- [x] 话术模板管理
- [x] 智能并发控制
- [x] 通话记录管理

- [x] NLU语义识别
- [ ] 拨打统计报表

- JDK 21 
- Maven 3.9
